Data Persistence Info

⚠ Render Free Tier: SQLite databases are stored inside the container's ephemeral filesystem. Data is wiped on each redeploy or after prolonged inactivity (usually 15 min spin-down). Product images are stored in localStorage (browser-side), so they persist across sessions on the same device.

✓ Fix (no code needed): Set up UptimeRobot (free) to ping your 4 Render URLs every 5 minutes to prevent spin-down. For persistent data, migrate to a hosted PostgreSQL database (Render, Supabase, or Neon — all have free tiers).
